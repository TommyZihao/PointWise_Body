# PointWise：气贯AI——中医人工智能图像识别

本项目仅用于科普教学和科学研究，识别结果仅供参考，任何后果与开发团队无关。

## 概述

基于人工智能、深度学习、计算机视觉、大语言模型技术，实现耳朵、舌面、手部、脚部、胸部、背部、腿部的中医穴位定位、语义分割和识别，用于辅助诊断。
